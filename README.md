# Apple StandBy Web (苹果待机显示 · Web版)

一个使用 **HTML + CSS + JavaScript** 实现的「苹果待机显示（StandBy）」风格网页  
👉 **可直接添加到 iOS 主屏幕运行**，横屏时作为待机时钟使用。

灵感来源于 iOS 官方 StandBy 模式，本项目仅用于学习与视觉还原。

---

## ✨ 功能特点

- 🕒 大号数字时钟显示（时 / 分 / 秒）
- 📅 显示完整日期（星期 + 年月日）
- 🌙 深色背景，适合夜间 / 待机使用
- 📱 支持 **iOS Safari 添加到主屏幕**
- 🔄 实时刷新，无需网络
- 🎨 高度还原苹果 StandBy 风格

## 🚀 使用方法

###  在线使用（推荐）

1. 使用 **iPhone Safari** 打开网页  
2. 点击「分享」
3. 选择 **添加到主屏幕**
4. 从主屏幕打开，即可全屏运行

> 建议锁定横屏使用，效果更佳
